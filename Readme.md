
# Landing Page Project

This project is a simple landing page designed using HTML and CSS Flexbox layout properties. The main purpose of this project is to make an entire layout only using Flexbox properties ‚≠ê‚≠ê‚≠ê.




## Features

This landing page template includes the following sections:

1. Header section:

- Logo (an icon)
- Navigation bar
- Greeting text
- Profile image

2. Main section:

- A heading introducing the content
- Image cards with associated text

3. Quote section:

- A beautifully styled inspirational quote

4. Newsletter Section:

- A call-to-action encouraging users to subscribe to a newsletter

5. Footer:

- Copyright info

## Technologies used

- HTML5: For structuring the content.
- CSS3: For styling and layout using Flexbox properties.
